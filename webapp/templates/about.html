ml4paleo is a collaborative open-science project.

The goal of this project is to develop simple and powerful machine learning models that can be used for paleontological research.

The Science

The following research has been completed thanks in part to the ml4paleo project:

Automated 3D image segmentation of synchrotron scanned fossils. Melanie A. D. During, Fredrik K. Gustafsson, Jordan K. Matelsky, et al. (In preparation, 2023)

The Code

All code is open-source and licensed under an Apache 2.0 license. The code is available on GitHub at @j6k4m8/ml4paleo.
